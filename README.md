# Usage
<!-- usage -->
```sh-session
$ npm install -g qdx
$ qdx COMMAND
running command...
$ qdx (-v|--version|version)
qdx/0.0.0 darwin-x64 node-v10.13.0
$ qdx --help [COMMAND]
USAGE
  $ qdx COMMAND
...
```
