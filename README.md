# Usage
<!-- usage -->
```sh-session
$ npm install -g qdx
$ qdx COMMAND
running command...
$ qdx (-v|--version|version)
qdx/0.0.0 darwin-x64 node-v10.13.0
$ qdx --help [COMMAND]
USAGE
  $ qdx COMMAND
...
```

## `qdx help [COMMAND]`

```
display help for <%= config.bin %>

USAGE
  $ qdx help [COMMAND]

ARGUMENTS
  COMMAND  command to show help for

OPTIONS
  --all  see all commands in CLI
```
